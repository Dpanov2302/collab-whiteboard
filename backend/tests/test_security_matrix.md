# Security and ownership matrix

Эти сценарии используются как ручной и автоматизируемый чеклист перед сдачей:

| Сценарий | Ожидаемый код |
|---|---|
| Гость вызывает `GET /api/workspaces` | 401 |
| Пользователь меняет чужую доску | 403 |
| Viewer создаёт элемент на доске | 403 |
| User вызывает `GET /api/admin/users` | 403 |
| Невалидный JWT | 401 |
| Устаревшая версия элемента при PATCH | 409 |
| Неверная ширина элемента `width=0` | 422 |
